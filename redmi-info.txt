# Redmi Project Reference

This file contains all necessary information and instructions
related to the OS_Updater project and future Redmi-specific automation.

---

## OS_Updater Scripts Reference

### Linux Updater
- File: linux-updater.sh
- Usage: chmod +x linux-updater.sh && ./linux-updater.sh
- Features: Updates packages, cleans unnecessary files, logs actions

### Windows Updater
- File: windows-updater.ps1
- Usage: Set-ExecutionPolicy Bypass -Scope Process -Force
         .\windows-updater.ps1
- Features: Updates Windows, cleans temp files, logs actions
